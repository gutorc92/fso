Sistema operacional e ambiente

Foi utilizado distribuições linux pelos dois alunos, especificamente o SO debian. Para compilação do código foi utilizado gcc, tal como consta todos makefiles, como editor de texto foi utilizado vim e Atom. Os tutoriais abaixo só foram testados em distribuições debian.

Na questão 3 foi utilizada a linguagem Python e uma biblioteca chamada numpy para ajudar na geração de matrizes. No README.md da pasta existe uma descrição de como se instalar a biblioteca.

Instruções de uso

Questão 1

Entre na pasta q1 pelo terminal e execute o comando:

$ make

Seguido de:

$ ./main

O programa irá imprimir o pid dos processos criados, dormir por 5 segundos, sinalizar o disparo de um sinal gerado pelo processo filho, e imprimi-lo na tela quando o processo pai capturar esse sinal. Então o programa irá esperar CTRL+C para finalizar sua execução.

Questão 2

Essa questão é composta por três soluções discutidas no README.md em sua pasta. Abaixo será descrito como executar cada uma.

Entre na pasta da q2 pelo terminal e execute o comando:

Para o código do executável q2

$ make

Para o código do executável q2wt

$ make -f Makefile_without_thread

Para o código do executável q2op

$ make -f Makefile_optimized


Agora basta executar seguindo a lógica:

$ ./q2 n x[0] x[1] x[2]

Onde x é um array que representa uma sequência numérica e n é o tamanho desse array.

Os resultados dessa questão, bem como outras analises da questão, estão no README.md nessa mesma pasta.

Questão 3


Entre na pasta da q3 pelo terminal e execute o comando:

$ make

Esse make irá gerar 3 executáveis: a, b e c. Para executa-los use a seguinte lógica:

$ ./a op matrix1.txt matrixs2.txt

Tal que op: t - mostra o tempo de execução, g - mostra a matriz resultante. Os argumentos matrix.txt são 2 arquivos textuais informando a dimensão da matriz na primeira linha e suas células nas linhas seguintes.

A execução em t irá exibir um número em microsegundos(10^6). Os resultados e a análise desse experimento está no arquivo README.md nessa mesma pasta q3.
