Sistema operacional e ambiente

Foi utilizado distribuições linux pelos dois alunos, especificamente o SO debian. Para compilação do código foi utilizado gcc, tal como consta todos makefiles, como editor de texto foi utilizado vim e Atom. Os tutoriais abaixo só foram testados em distribuições debian.

Instruções de uso

Questão 1

Pelo terminal execute o comando:

$ make

Seguido de:

$ ./prog number(um número de 1 a 9)

Logo irá aparecer uma sequência de letras sendo impressas. Digite após algumas dessas impressões, CTRL + C, irá finalizar a sequência de impressões e aparecer estatisticas sobre as threads que foram executadas na impressão das letras.
