Sistema operacional e ambiente

Foi utilizado distribuições linux pelos dois alunos, especificamente o SO debian. Para compilação do código foi utilizado gcc, tal como consta todos makefiles, como editor de texto foi utilizado vim e Atom.

Instruções de uso

Questão 1

Entre na pasta da q1 pelo terminal e execute o comando:

$ make

Seguido de:

$ ./prog

Logo irá aparecer uma tela requisitando pontos no eixo cartesiano. Após digitar todos pontos ele irá te informar se essas coordenadas podem formar um triangulo. Caso forme ele exibe a área desse triangulo e seu perimetro.
Para executar alguns casos de teste definidos, use:

./prog < test
