Sistema operacional e ambiente

Foi utilizado distribuições linux pelos dois alunos, especificamente o SO debian. Para compilação do código foi utilizado gcc, tal como consta todos makefiles, como editor de texto foi utilizado vim e Atom. Os tutoriais abaixo só foram testados em distribuições debian.

Instruções de uso

Questão 1

Entre na pasta da q1 pelo terminal e execute o comando:

$ make

Seguido de:

$ ./prog

Logo irá aparecer uma tela requisitando pontos no eixo cartesiano. Após digitar todos pontos ele irá te informar se essas coordenadas podem formar um triangulo. Caso forme ele exibe a área desse triangulo e seu perimetro.
Para executar alguns casos de teste definidos, use:

./prog < test

Questão 2

Entre na pasta da q2 pelo terminal e execute o comando:

$ make

O executável irá ordenar números passados por argumento em sua execução, para isso execute:

$ ./prog numero1 numero2 numero3...

Para qualquer número inteiro. Você pode usar antes dos números os comandos -d e -r ainda, tal -d a ordem será crescente e -r a ordenação será decrescente.

Questão 3

Para essa questão foi criado um README que responde as duas perguntas de seu enunciado. Olhar a pasta q3.
Entre na pasta da q3 pelo terminal e execute o comando:

$ make

Em seguida execute:

$ ./main

Primeiramente será impresso algumas linhas, seguidos de requisição de texto, digite algum texto sem espaço, tabulaçao ou quebra de linha e ele ira imprimir o valor de string 2 e depois o seu valor concatenado em outra string.
